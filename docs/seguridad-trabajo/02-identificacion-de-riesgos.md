# Identificacion de Riesgos y Medidas de Control

## Objetivo
Definir un proceso practico para identificar peligros, evaluar riesgos y aplicar controles efectivos en cada puesto de trabajo.

## Metodologia sugerida
1. Identificar tareas y actividades criticas.
2. Detectar peligros por categoria.
3. Evaluar probabilidad e impacto.
4. Definir controles segun jerarquia.
5. Verificar eficacia y ajustar.

## Tipos de peligros frecuentes
- Fisicos: ruido, vibraciones, temperaturas extremas, iluminacion deficiente.
- Quimicos: polvo, vapores, solventes, gases.
- Biologicos: bacterias, virus, hongos.
- Ergonomicos: posturas forzadas, movimientos repetitivos, carga manual.
- Psicosociales: estres, exceso de carga, turnos extensos.
- Mecanicos: partes moviles, herramientas defectuosas, golpes y atrapamientos.

## Jerarquia de controles
1. Eliminacion del peligro.
2. Sustitucion por una alternativa menos peligrosa.
3. Controles de ingenieria (barreras, ventilacion, resguardos).
4. Controles administrativos (procedimientos, permisos, rotacion).
5. Equipo de proteccion personal (EPP).

## Buenas practicas
- Realizar inspecciones de rutina por area.
- Incluir al personal operativo en la identificacion de riesgos.
- Registrar casi accidentes para prevenir eventos mayores.
- Actualizar la matriz de riesgos tras cambios operativos.
